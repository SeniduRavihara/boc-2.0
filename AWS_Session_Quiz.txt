1. What is cloud computing?
A. Using only local computers
B. Using internet to access computing services
C. Using mobile apps only
D. Using hardware devices
■ Answer: B

2. Which is an example of cloud storage?
A. Hard disk
B. USB drive
C. Google Drive
D. RAM
■ Answer: C

3. What is the main benefit of cloud computing?
A. More hardware needed
B. Flexibility and cost saving
C. No internet
D. Slow performance
■ Answer: B

4. Which concept allows multiple virtual machines to run on one physical server?
A. Load balancing
B. Virtualization
C. Encryption
D. Compression
■ Answer: B

5. A company combines local servers with cloud services. What is this called?
A. Public cloud
B. Private cloud
C. Hybrid cloud
D. Community cloud
■ Answer: C

6. Which AWS service is used to control user access and permissions?
A. Amazon EC2
B. Amazon S3
C. AWS IAM
D. Amazon RDS
■ Answer: C

7. Which AWS service is used to run code without managing servers?
A. Amazon EC2
B. AWS Lambda
C. Amazon S3
D. Amazon RDS
■ Answer: B

8. Which AWS service is used for a managed relational database?
A. Amazon S3
B. Amazon EC2
C. Amazon RDS
D. Amazon CloudFront
■ Answer: C

9. Which AWS service provides a private network in the cloud?
A. Amazon S3
B. AWS Lambda
C. Amazon EC2
D. Amazon VPC
■ Answer: D

10. Which AWS service is used for content delivery (CDN)?
A. Amazon EC2
B. Amazon RDS
C. Amazon CloudFront
D. AWS Lambda
■ Answer: C

11. What is an AWS Region?
A. A virtual machine
B. A geographic area with data centers
C. A storage system
D. A database
■ Answer: B

12. What is an Availability Zone?
A. A programming tool
B. A separate data center in a region
C. A mobile app
D. A storage device
■ Answer: B

13. What does "pay-as-you-go" mean?
A. Fixed cost
B. Pay only for what you use
C. Free always
D. Pay before use
■ Answer: B

14. What is scalability in cloud computing?
A. Deleting data
B. Increasing or decreasing resources
C. Writing code
D. Printing files
■ Answer: B

15. What is high availability?
A. System always offline
B. System runs with minimal downtime
C. No internet needed
D. Low storage
■ Answer: B

16. In the Shared Responsibility Model, who secures the physical data center?
A. Customer
B. AWS
C. Both
D. No one
■ Answer: B

17. In the Shared Responsibility Model, who manages application security?
A. AWS
B. Customer
C. Both
D. No one
■ Answer: B

18. Why deploy applications across multiple Availability Zones?
A. Reduce cost
B. Improve fault tolerance
C. Increase storage
D. Simplify coding
■ Answer: B

19. A system automatically adjusts resources based on demand. What is this called?
A. Virtualization
B. Elasticity
C. Encryption
D. Backup
■ Answer: B

20. A developer exposes sensitive data by misconfiguring a service. What is the issue?
A. AWS failure
B. Network issue
C. User misconfiguration
D. Hardware failure
■ Answer: C
